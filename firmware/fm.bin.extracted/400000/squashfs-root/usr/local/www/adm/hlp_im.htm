Help - HTTP

HTTP Notification
URL Enter the Uniform Resource Locator (URL) of your HTTP notification server.
User Name Enter your user name for the HTTP notification server.
Password Enter your password for the HTTP notification server.
Proxy Server Name
(optional)		 In order to connect your HTTP notification server indirectly, you need to specify the proxy server name here.
Proxy User Name
(optional)		 Enter your user name for the Proxy Server.
Proxy Password
(optional)		 Enter your password for the Proxy Server.
Proxy Port Number
(optional)		 Enter the port number here when you make an indirect network connection through a proxy server.
Method Select the desired method of form data encoding.  The default method is set to "GET".
"GET" should be used if and only if the form processing is idempotent, which typically means a pure query form. Generally it is advisable to do so.
However, if there are problems related to long URLs and non-ASCII character repertoires, which can make it necessary to use "POST" even for idempotent processing.
With Payload Select this check box if there are payload to post and method is POST.
