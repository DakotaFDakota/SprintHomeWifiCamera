Help - IP Filter
  • The IP Filter feature allows administrators to control network camera access by filtering IP address. These restrictions apply both to traffic on the WAN port and local (LAN) traffic.
  • If there is no need to set this restriction, simply remain or select "Disable" - Factory Default.
  • Or, you can select either deny or allow to those IP address that you manually add on your filtering list.

IP Filter
IP Filter Select to perform the desired IP address(or addresses) filtering function.

Filtering List   -   Maximum: 10
Single / Range Select to perform either single IP address or a range of IP addresses that you desired.
(Single) IP Address Enter an IP address you would like to allow or deny.
Start IP ~ End IP Enter a range of IP addresses you would like to allow or deny.
