Help - FTP

FTP Server(Primary & Secondary)
FTP Server Enter the address of the Primary/Secondary FTP Server.
Port Enter the Port of the Primary/Secondary FTP Server to be connected.
Login name Enter the Login name for the Primary/Secondary FTP server.
Password Enter the password for the Primary/Secondary FTP server.
Passive mode Check the box to enable the Passive mode feature of the Primary/Secondary FTP.
File Path name Enter the file path/name of the Primary/Secondary FTP.
Test the Server Test the server settings with the server.
Enable Secondary FTP Check the box to upload to the secondary FTP if the camera can not connect to the primary FTP.
