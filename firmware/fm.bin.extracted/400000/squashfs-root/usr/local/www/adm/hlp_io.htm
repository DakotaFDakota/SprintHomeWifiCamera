Help - I/O Ports

This page allows configuration of the input port and output port supported by the Network Camera. Port's triggering condition can be configured.

Input Port
Current State This indicates the current status of input device (ex. Alarm Sensor or Smoke Sensor) that you connected.
Triggering Condition There are High, Low, Rising, and Falling - 4 triggering conditions can be set. The input port will be activated if the triggering condition changes so that it no longer equals to the current State.

Output Port
Current and
Default State		 In the Network Camera, as long as both output's Default and Current State is High, the output is disconnected (floating). When the current State change to Low or Low Pulse, the output will be activated.

The default Default State for output on the Network Camera is High. While the Network Camera is restarting, the output will be in the Default State (i.e. High) for at least as long as it takes for the Network Camera to fully initialize. Please consider this carefully when connecting equipment to the output. After a completed restart, the unit will then switch to Low or Low Pulse (but only if configured to do so.)
Manual Trigger Manual trigger action type is depends on output's default State. Changing and saving Action Definition setting. When manually trigger, the state will be changed as you configured.
Action When Triggered

Changing and saving Action When Triggered settings on this page will also reset output to their Default States. The output state will be correct on the next time when triggering event activates the output.

Pin Definition
Pin No. Function Description
01 DI Photo Coupler Input:
Isolated digital input port which can be used to connect DI/GND.
Connect to DI GND to activate or leave floating (or unconnected) to deactivate.
02 GND DI GND
03 DO Open Collector Output:
Type: NPN transistor.
Collector (Pin3) to emitter input voltage (Pin4) :+24VDC.
Collector current: 100mA.
This output has an open collector NPN transistor with the emitter connected to GND (Pin4). If it is to be used with an external relay, a diode must be connected in parallel with the load for protection against any voltage transients.
