Help - SMB Client

SMB/CIFS Client
Browse SMB/CIFS Server Click to search for an available SMB/CIFS server on your network. And you also need to select a folder on the selected server to store the recording video.
Server Name Enter the server name of your SMB host server.
File Path Enter the file path of your SMB host server.
User Name Enter your user name for the SMB client account.
Password Enter your password for the SMB client account.
Test the Server Test the server settings with the server.

Continuous Recording
Maximum duration Maximum duration of continuous recording file and unit is Minutes, range is 1 to 60.
Browse SMB/CIFS Server Click to search for an available SMB/CIFS server on your network. And you also need to select a folder on the selected server to store the recording video.
Server Name Enter the server name of your SMB host server for continuous recording.
File Path Enter the file path of your SMB host server for continuous recording.
User Name Enter your user name for the SMB client account for continuous recording.
Password Enter your password for the SMB client account for continuous recording.
Test the Server Test the server settings with the server for continuous recording.
