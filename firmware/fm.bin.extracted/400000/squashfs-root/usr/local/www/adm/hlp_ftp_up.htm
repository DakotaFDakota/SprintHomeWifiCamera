Help - FTP Upload

An FTP upload is a set of parameters describing how and when the Network Camera will upload video or snapshot files to a FTP server.

Notes:

  • It is important to note that adding a new schedule will stop any upload that is currently running. A scheduled upload will automatically resume, if the time is still within the scheduled period. A triggered upload will need to be re-triggered.
  • The Network Camera can be configured for up to 10 FTP upload schedules.

Scheduled Upload
Enable Check to perform all of the upload(s) that were configured and scheduled.
Interval Select the desired option for the uploads interval.
Scheduled Upload The Schedule lists all of the uploads currently configured in the Network Camera, along with various information about their configuration, as listed below:
  • Name - the descriptive name of upload event set by the user
  • Effective Time Frame - shows when the upload at a set time will be triggered
  • Attachment - shows what kind of file will be attached when the upload been triggered

Add New Schedule
Name Provide a descriptive name for the upload event, using a maximum of 20 characters.
Effective Time Frame Choose the desired option for the period.
Start Time Choose the desired start time using a 24 hr clock.
End Time Choose the desired end time using a 24 hr clock.

Attachment
Attachment Type JPEG Image - (MJPEG Mode Only)
Snapshot Select the desired photo shot(s) here.
Interval Select the desired option for the upload interval.

Video Clip - (MPEG-4 Mode Only)
File Format Select the desired format for the video file.
Video Length Select the desired length for the video clip.
