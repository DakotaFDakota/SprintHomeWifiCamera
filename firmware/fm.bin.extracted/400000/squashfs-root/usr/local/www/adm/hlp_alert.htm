Help - E-Mail

SMTP Server (Primary & Secondary)
SMTP Server Address Enter the address of the SMTP (Simple Mail Transport Protocol) Server to be used to send E-Mail.
Port Enter the Port of the Primary/Secondary SMTP Server to be connected.
Authentication Select the desired Authentication type for the SMTP Server.
SMTP Login name Enter your login name for the SMTP Server.
SMTP Password Enter your password for the SMTP Server.
POP server name Enter the name for the POP Server.
Show "From" as Enter the E-Mail address to be shown in the "From" field when the E-Mail is received.
Test the Server Test the server settings with the server.
Enable Secondary SMTP Check the box to upload to the Secondary SMTP if the camera can not connect to the primary SMTP.

E-Mail Setup
E-Mail Address For E-Mail alerts to function correctly, you must enable and enter at least one (1) E-Mail address. You can use the individual checkboxes to enable or disable each E-Mail address as required.
Subject Enter the desired text to be shown as the "Subject" for the E-Mail when it is received.
With Attachment If checked, the sent out E-Mail will include an attachment. (The attached file settings need to be configured on the "Event Trigger" page.)
