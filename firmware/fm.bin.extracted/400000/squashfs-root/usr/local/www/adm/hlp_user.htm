Help - User Database

Existing Users
User List This displays all users you have entered into the User database. If you have not entered any users, this list will be empty.
Button Use the Edit, Delete and Delete All buttons to manage the user database.

Add New User
User Name
  • Enter the name for the user here. Spaces, punctuation, and special characters must NOT be used in the name.
  • The name is case insensitive (case is ignored), so you can not have 2 names which differ only by case.
User Password The password for the user.
Confirm Password Re-enter the password for the user, to ensure it is correct.
Control Level Select the desired control level for the user you plan to add. (* Operator level allows user to control Pan/Tilt)
Buttons Use the Add and Clear buttons to manage the user properties.
