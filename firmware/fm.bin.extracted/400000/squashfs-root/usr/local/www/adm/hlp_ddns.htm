Help - DDNS

DDNS

By using a DDNS (Dynamic DNS) Service, users can connect to your Server using a Domain name even if you have a dynamic IP address. When your dynamic IP address changes, the DDNS client embedded in this Server will automatically update the servers' name that host your DNS records.

  • If your Router/Gateway supports DDNS, you should use the DDNS feature in the Router/Gateway, rather than this one.
  • You do NOT need to use DDNS "client" software on your PC to perform IP address updates, since updates are performed by this server.

DDNS Settings
Enable DDNS Enable or disable the DDNS function, as required.
Only enable this feature if you have registered for the DDNS Service with a DDNS Server provider.
Service Provider Choose a service provider from the list.
Web Site
Button		 Click this button to open a new window and connect to the Web site for the selected DDNS service provider.
Domain (Host) Name Enter the Domain Name (host name) allocated to you by the DDNS Server provider.
Account Enter your login name for the DDNS account.
Password Enter your password for the DDNS account.
Check WAN IP Address Set the schedule for checking if the Internet IP address has changed. If the IP address has changed, the DDNS Server will be notified.

NOTE: If the DDNS Service provided some software to perform this IP address update or notification, you should NOT use this software. The update is performed by the camera.
