Help - Config Event Trigger

Name: Event configuration name.

Schedule
Always Always enable this event.
Schedule Enable this event by schedule.
Never Disable this event.

Trigger
Motion Detection Trigger by motion detection.
PIR Trigger by PIR.
Input Trigger from input port.
Audio Detection Trigger from audio detection.
HTTP CGI Trigger from http_trigger.cgi.
Periodically Trigger periodically.
Continuously Trigger continuesely.
Interval Select the desired option for the events interval. (* "0" = No Delay)

Action
Output Port If checked, the output port state will be activated as configured. (* Output port must first be configured on the I/O Port page.)
White Light LEDs If checked, White Light LEDS will be turn on during night mode.
HTTP If checked, an HTTP notification will be delivered to the HTTP server. (* HTTP server must first be configured on the HTTP page first.)
E-Mail If checked, an E-Mail (with "Attachment") will be delivered to the SMTP server. (* SMTP Server must first be configured on the E-Mail page.)
FTP If checked, an FTP upload (with "Attachment") will be activated to the FTP server. A secondary server may also be specified, if required. This will be used if the primary server is unavailable.   (* FTP servers must first be configured on the FTP page.)
SMB/CIFS If checked, video files will be uploaded to the SMB/CIFS host server. (* SMB/CIFS client must first be enabled and configured on the SMB/CIFS Client page first.)
SMB Recording If checked, video files will be recorded to the SMB/CIFS host server. (* SMB/CIFS client must first be enabled and configured on the SMB/CIFS Client page first.)
Pre-Capture Select the desired length for the video and audio before event trigger.
Post-Capture Select the desired length for the video and audio after event trigger.
