Help - SD Card

Card Management
Card Status This indicates the current status of the SD card (ex. N/A or Read and Write).
Refresh Refresh the information of SD card.
Format Format the SD card to FAT32.
Safely Remove Press this button before you unplug the SD card from camera.
View View clips on SD card.

Recording
Enable Check to perform the recording function.
Duration Set the maximum time of recorded clips.
Overwrite/Stop Select the desired action when the SD card is full.

Schedule
Recording Schedule This displays all time periods you have configured in the database. The list is empty if you haven't configured any schedule.
Delete Click this button to remove the selected item from the list.
Day Choose the desired option for the period.
Start Time Enter the start time using a 24 hr clock.
End Time Enter the end time using a 24 hr clock.
Add Click this button to add a new period.
Clear Click this button to clear period.
