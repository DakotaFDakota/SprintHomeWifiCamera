Help - Event

An trigged event is a set of parameters describing how and when the Video Server will perform certain triggered actions, according to requirements.

  • It is important to note that adding a new event will stop any event that is currently running. A scheduled event will automatically resume, if the time is still within the scheduled period. A triggered event will need to be re-triggered.
  • The Video Server can be configured for up to 10 events.

Settings
Enable Check to perform all of the event(s) that were configured.
Add Add a new event to the event confiuration table.
Edit Edit the configuration of the selected event.
Delete Delete a event from the event confiuration table.

Motion Position
Enable Motion If select, enable Motion Detectio in all position.
Disable Pan/Tile If select, disable Pan/Tilt while Motion Detection is enabled.
Disable Motion If select, disable Motion Detection while current position is incorrect, and can set go to Motion Detection position wait minutes.
idle time If select this check box, Go to Motion Detection position if idle some minutes.
