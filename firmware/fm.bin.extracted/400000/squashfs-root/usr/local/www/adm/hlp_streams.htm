Default Streaming Channel
Streaming Channel When accessing the "View Video" page, the client will use the selected streaming channel to view the video.

Streaming 1/2/3 Settings
Enable Only for Streaming 2/3:
Enable the streaming 2/3.
Video Format Select a video format that uses for video streaming.
Resolution Select the desired video resolution format.
Fix Video Quality Only for MJPEG:
Select the desired fix quality. The default fix quality is set to Normal.
Note: Higher image quality requires more bandwidth.
Video Quality Control Only for H264:
  • Constant Bit Rate: Select the desired fix bit rate.
  • Fix Quality: Select the desired fix quality. The default fix quality is set to Normal.

    • Note: Higher image quality requires more bandwidth.
    GOV Length Only for H264:
    Adjust the GOV interval in frame base.
    Max Frame Rate Select the desired maximum frame rate. The default maximum frame rate is set to 30 fps.  
    Note: Higher frame rate requires more bandwidth.
    User Defined URI Setup a specific URL for accessing the live video from Network Camera without authentication check. It usually uses for cell phone connections.
    Cropping Select "None" for scaling, select "From Streaming 1" or "From Streaming 2" for croping the video from streaming 1/2
