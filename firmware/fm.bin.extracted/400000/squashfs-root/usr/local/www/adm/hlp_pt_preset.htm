Help - Preset Point Position

Preset Point Position
Calibration Button Click this button to reset the calibration of Pan/Tilt area.
Set Calibration Home Set Lens back to the default location.
Preset List Select the desired preset position, The screen will update with the current data for the selected Preset.
If you don't need the desired preset position, click the "Remove" button to remove.
Preset Name Enter a suitable name for the current desired position and then click the "Add" button to add into the preset list.
