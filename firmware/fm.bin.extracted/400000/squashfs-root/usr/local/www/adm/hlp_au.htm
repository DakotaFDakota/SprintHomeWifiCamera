Help - Audio Detection

Set Detection Volume
Current Volume The reference level of the current environment volume.
Refresh Button Click to get the current environment volume level.
Triggered Volume Adjust the triggered level to cause an audio event alert.
Triggered When
  • Low to high - If selected, the audio alert is triggered when the current volume rises above the triggered volume level.
  • High to low - If selected, the audio alert is triggered when the current volume falls below the triggered volume level.
